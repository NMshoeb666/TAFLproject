The Pumping Lemma for Regular Languages

If a language L is regular, then there exists a number p (the pumping length) such that any string s ∈ L with |s| ≥ p can be split into three parts, s = xyz, satisfying:

  • |y| > 0 (y is not empty)
  • |xy| ≤ p (the first two parts are within the pumping length)
  • For all i ≥ 0, the string xyⁱz ∈ L

PROOF BY CONTRADICTION

To prove a language is not regular:

  1. Assume the language is regular.
  2. Show that for any choice of p, we can find a string s that cannot be "pumped" without leaving the language.
  3. This contradiction proves the initial assumption was false.
